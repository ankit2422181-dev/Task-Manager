# Task-Manager
Backend-based task management system with authentication and priority-based scheduling using Node.js and MongoDB
# Smart Task Management System

## Overview

A backend-based task management system built using Node.js and Express that allows users to create, update, delete, and prioritize tasks efficiently.

## Tech Stack

* Node.js
* Express.js

## Features

* Create, update, and delete tasks (CRUD operations)
* Priority-based task sorting
* REST API architecture
* Modular project structure (routes and models)

## Key Highlight

Implemented priority-based sorting logic to optimize task handling and improve efficiency.

## Future Improvements

* Add MongoDB database for persistent storage
* Implement user authentication (JWT)
* Add advanced filtering and scheduling features
